# Sprint 1 Plan

## Sprint Goal
Implement the core features that allow residents to register, log in, and submit complaints, and allow barangay officers to view and update them.

## Sprint Duration
- **Start Date:** April 24, 2026
- **End Date:** May 8, 2026

## Selected User Stories for Sprint 1

| # | User Story | Story Points | Assigned To |

| US-01 | As a resident, I want to submit a complaint online so that I don't have to go to the barangay hall in person. | 5 | Austria |
| US-02 | As a resident, I want to track the status of my complaint so that I know if it's being addressed. | 3 | Austria |
| US-03 | As a barangay officer, I want to view all submitted complaints so that I can manage them efficiently. | 5 | Saguilayan |
| US-04 | As a barangay officer, I want to update the status of a complaint so that residents are informed of progress. | 3 | Saguilayan |
| US-05 | As a resident, I want to register an account so that my complaints are linked to my identity. | 3 | Oga |
| US-06 | As a resident, I want to log in securely so that only I can see my complaints. | 2 | Oga |

## Total Story Points: 21

## Sprint Notes
- Focus on basic functionality first, no need for fancy UI in Sprint 1.
- Daily check-ins via group chat to update progress.
- All features should be tested before marking as done.
